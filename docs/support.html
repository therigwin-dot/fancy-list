üîç Common Issues and Solutions

Web Part Not Loading

  • Check that the web part is properly deployed to your SharePoint environment
  • Verify that you have the necessary permissions to add web parts to pages
  • Try refreshing the page or clearing browser cache

List Data Not Displaying

  • Ensure the selected list exists and contains data
  • Verify that you have read permissions for the list
  • Check that the Category, Subject, and Description fields are properly configured

Styling Not Applied

  • Check that the property pane settings are saved
  • Try refreshing the page to see if changes take effect
  • Verify that custom CSS is not conflicting with the web part styles

Filter Not Working

  • Ensure the Category field is properly selected in the configuration
  • Check that the list contains data in the Category field
  • Verify that the filter section is enabled in the property pane
